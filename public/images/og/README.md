# OGP Images

SNS共有用の画像置き場です。

現在の参照先は `data/siteConfig.ts` の `ogImage` です。

## 現在使っている画像

- `rock-bar-heaven-og.jpg`

## 差し替える時の推奨

- ファイル名は `rock-bar-heaven-og.jpg` のまま差し替える
- 推奨サイズは `1200x630`
- `app/layout.tsx` のOGP metadataも `1200x630` 前提です
- 黒基調、赤坂、地下感、深夜感、ロックバー感が伝わる写真またはデザイン
- 文字を入れる場合は `ROCK BAR HEAVEN AKASAKA` と `赤坂` までに絞る
- フリー素材風、綺麗すぎるホテルバー風、テンプレート風は避ける
